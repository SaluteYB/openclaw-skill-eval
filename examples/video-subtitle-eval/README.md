# Example: video-subtitle evaluation pack

This folder shows what a small but real evaluation pack can look like when using `skill-eval`.

Target skill: `video-subtitle`

Purpose:
- verify that subtitle-file requests route to `video-subtitle`
- keep ffmpeg/media-export tasks routed away from it
- document boundary cases around transcript preparation

Files in this folder:
- `should-trigger.md`
- `should-not-trigger.md`
- `boundary-cases.md`
- `execution-checklist.md`
- `evaluation-log.md`
